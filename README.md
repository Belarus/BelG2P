Фанетычны канвертар беларускай мовы
===================================

Фанетычны канвертар канвертуе беларускія словы ў іх фанетычнае прадстаўленне.

Выкарыстанне канвертара:

	GrammarDB2 db = GrammarDB2.initializeFrom...  // чытаць Граматычную базу беларускай мовы
	GrammarFinder finder = new GrammarFinder(db); // стварыць пошукавік па Граматычнай базе
	FanetykaConfig config = new FanetykaConfig(); // стварыць канфігурацыю на падставе пошукавіку і табліц канвертара
	IFanetyka f = new Fanetyka3(config);          // стварыць канвертар
	f.addWord(word); f.addWord(...); ...          // дадаць словы для канвертавання
	f.calcFanetyka();                             // канвертаванне
	f.toString(Huk.ipa);                          // вывад фанетыкі ў патрэбным стандарце
